# Portfolio Projects

A collection of **quantitative finance**, **machine learning**, and **data science** projects showcasing real-world problem solving using Python, SQL, and statistical analysis.

## 📊 Projects Overview

### 1. **A/B Testing at World Quant University**
Statistical hypothesis testing and experimental design for financial applications.
- **Skills**: A/B testing, statistical power analysis, hypothesis testing, chi-square tests
- **Tools**: pandas, numpy, scipy, statsmodels, MongoDB, Plotly
- **Key Concepts**: Statistical significance, experiment design, test validation

### 2. **Buenos Aires Real Estate Analysis** (Tasks 1-4)
Multi-phase real estate price prediction and analysis project.
- **Skills**: Regression modeling, data cleaning, feature engineering, model evaluation
- **Tools**: scikit-learn, pandas, feature selection techniques
- **Key Tasks**:
  - Task 1-2: Data preparation and exploratory analysis
  - Task 3-4: Linear regression modeling and predictions

### 3. **Credit Risk Analysis in the US**
Clustering and segmentation analysis of credit risk patterns.
- **Skills**: Unsupervised learning, clustering (K-Means), dimensionality reduction (PCA)
- **Tools**: scikit-learn, seaborn, matplotlib, pandas
- **Key Methods**: Silhouette scoring, standardization, pipeline construction

### 4. **Earthquake Damage Prediction in Nepal**
Classification model for predicting building damage severity.
- **Skills**: Classification, feature encoding, imbalanced data handling, pipeline design
- **Tools**: scikit-learn, pandas, SQLite, seaborn, matplotlib
- **Key Methods**: One-hot & ordinal encoding, Logistic Regression, Decision Trees

### 5. **Brazil Real Estate Analysis - Mexico**
Real estate market analysis with data visualization and cleaning.
- **Skills**: Data cleaning, visualization, EDA, market analysis
- **Tools**: pandas, matplotlib, Plotly, unicodedata
- **Focus**: Handling missing data, geographic analysis

### 6. **Poland Bankruptcy Prediction**
End-to-end classification pipeline for company bankruptcy prediction.
- **Skills**: Classification, imbalanced dataset handling, model selection, hyperparameter tuning
- **Tools**: scikit-learn, imblearn, pandas, GridSearchCV
- **Key Methods**: Random oversampling, Random Forest classification, cross-validation

### 7. **Market Volatility in India**
Financial market analysis and volatility assessment.
- **Skills**: Time series analysis, financial metrics, market analysis
- **Tools**: pandas, matplotlib, statistical analysis
- **Focus**: Market trends, volatility patterns, risk assessment
